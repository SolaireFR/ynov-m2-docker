## Commandes Docker pour la gestion d'images Docker Morgan SECRETIN

### 1. Téléchargement d'une image (Nginx)
```bash
docker pull nginx
```

### 2. Liste d'images (Nginx)
```bash
docker images
```

### 3. Suppression d'une image (Nginx)
```bash
docker image remove nginx
```

### 4. Téléchargement de plusieurs images (MySQL et Redis)
```bash
docker pull mysql
docker pull redis
```

### 5. Liste d'images (Mysql et Redis)
```bash
docker images
```

### 6. Affichage détaillé d'une image (MySQL)
```bash
docker inspect mysql
```

### 7. Historique d'une image (MySQL)
```bash
docker history mysql
```

### 8. Recherche d'images (Php) (j'aime bien php ^_^)
```bash
docker pull php:7.4-apache
docker inspect php:7.4-apache
```
